# <a name="microsoft-intune-protected-apps"></a>Microsoft Intune aplicativos protegidos  

Os aplicativos listados neste tópico são parceiros com suporte e aplicativos da Microsoft que são comumente usados com Microsoft Intune. Os aplicativos protegidos do Intune são habilitados com um rico conjunto de políticas de proteção de aplicativo móvel. Esses aplicativos permitem que você:

- Restringir as funções de copiar e colar e salvar como
- Configurar links da Web para abrir dentro do navegador seguro da Microsoft
- Habilitar o uso de várias identidades e acesso condicional no nível do aplicativo
- Aplicar políticas de prevenção de perda de dados sem gerenciar o dispositivo do usuário
- Habilitar a proteção do aplicativo sem a necessidade de registro
- Habilitar a proteção de aplicativo em dispositivos gerenciados com ferramentas do EMM de terceiros

> [!NOTE]
> Para seus aplicativos de linha de negócios de cliente, você pode incorporar o gerenciamento de aplicativos móveis usando o SDK ( [Software Development Kit](../developer/app-sdk.md) ) do aplicativo do Intune ou a [ferramenta de disposição do aplicativo para IOS](../developer/app-wrapper-prepare-ios.md) e a [ferramenta de disposição do aplicativo para Android](../developer/app-wrapper-prepare-android.md).

As tabelas a seguir fornecem detalhes de parceiros com suporte e aplicativos da Microsoft que são comumente usados com o Microsoft Intune.
