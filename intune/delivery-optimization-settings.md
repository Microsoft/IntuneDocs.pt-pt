# <a name="delivery-optimization-settings-for-intune"></a>Definições de otimização de entrega para o Intune

Este artigo lista as definições de otimização de entrega que o Intune suporta para dispositivos que executam o Windows 10 ou posterior.  

A maioria das opções na consola do Intune são mapeadas diretamente para as definições de otimização de entrega que são explicadas em mais detalhes na documentação do Windows, para os quais são fornecidas hiperligações para conteúdo relevante.  Configurações ou opções específicas para o Intune não contêm ligações para conteúdo adicional.  

As tabelas seguintes incluem:  

- **Definição**: A definição como ele aparece no Intune. Definições de ligações de abrir a entrada relevante no [configurar a otimização de entrega para 10 Windows atualiza](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization) na documentação do Windows, onde pode saber mais sobre a definição.

- **Versão do Windows**: A versão mínima do Windows 10, que inclui suporte para esta definição.  

- **Detalhes**: Uma breve descrição de como o Intune implementa a definição, incluindo o predefinido do Intune. Se estiver disponível, existem links para [fornecedor de serviços de configuração de política de otimização de entrega](https://docs.microsoft.com/windows/client-management/mdm/policy-csp-deliveryoptimization) entradas (CSP).  

Para configurar o Intune para utilizar estas definições, consulte [fornecer atualizações](delivery-optimization-windows.md).
